Tutorial

Insira a folha de estilos

<style type="text/css" rel="stylesheet">
/* estilo principal */
body {
background-image: url('fantasy-girl-in-spring-wallpaper(www.CoolWallpapers.org).jpg');
background-position: 50% 0px;
background-attachment: fixed;
background-repeat: no-repeat;
}

article {
background-attachment: fixed;
background-repeat: no-repeat;
}
</style>

Chame o Javascript

<script type="text/javascript" src="jquery.min.js"></script>
<script type="text/javascript" src="StackBlur.js"></script>
<script type="text/javascript" src="parallax2.js"></script>

Diga como é seu CSS:

<script type="text/javascript" >
parallax2.configure(
{
// O post no CSS, exemplo article ou .post-data
articleSelector:'#content article',
// Onde vem a imagem de fundo, em geral é body mesmo
backgroundSelector:'body'
// Opcionalmente você pode desativar o scroll e ter mais controle no CSS
,disableScroll: true
} );
</script>
